# Instalação

Utilizar comando.

- sudo apt update

- sudo apt install apache2

Se der erro tentar a baixo a opção.

Antes da opção de Intall, solicitou para fazer a configuração.

- sudo dpkg --configure -a

depois rodar o intall.
